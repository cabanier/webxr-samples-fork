Inline Stereo Session

This sample demonstrates use of an inline XRSession with the 'inline-stereo' feature to present stereo WebXR content directly on the page. The optional 'local' feature enables tracked inline stereo frame data. The stereo panorama uses eye-specific source imagery, and the solar system model is rendered once for each XRView. If 'local' is not granted, the session still renders inline stereo output with an emulated viewer pose. The viewer pose can be adjusted by clicking and dragging with mouse or touch. Back

Inline Multi-View Session

This sample demonstrates use of an inline XRSession with the 'multi-view' and 'head-tracked' features to present stereo WebXR content directly on the page. The stereo panorama uses eye-specific source imagery, and the solar system model is rendered once for each XRView. If head tracking is not granted, the session still renders multi-view output without tracking to the user's head pose. The viewer pose can be adjusted by clicking and dragging with mouse or touch.
